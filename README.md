# Онлайн сервис "Помошник в Бизнесе"

Сервис предназначен для упрощения работы дистрибьютера (лицо занимающееся сетевым маркетингом), он позволяет получать обратную связь от потенциальных клиентов в том числе принимать заказы и оплату товаров онлайн. Создание личной страницы не требует навыков программирования. Адрес страницы имеет вид "1000000stores.ru/7111222333", где 7111222333 это номер телефона дистрибьютера.

официальный сайт проекта:  
http://1000000stores.ru/

## 1 обзор проекта

![1](https://github.com/2023samir/website-your-business-helper/blob/86d78641c53ee9d831c55f75885457daf1e7c6ab/README/1.JPG)

Функционал:

- Зарегистрированный на сайте пользователь(дистрибьютер) может:

- Разместить информацию о:

- предлагаемых пользователем товарах
- Методах обучения его потенциальных рекрутов

- Получать обратную связь от потенциальных клиентов:

- - Заказы товаров
- - Заявки от потенциальных рекрутов
- - Запросы обратного звонка
- - Заявки на получение рассылки

- Зарабатывать воспользовавшись партнерской программой сервиса

- Не зарегистрированный пользователь может:

- На главной странице:

- - Найти всех дистрибьютеров указанной продукции и компании в указанном городе

- на странице дистрибьютера:

- Ознакомиться с профилем владельца страницы (ИНН, телефон, Skype, email)
- Заказать обратный звонок
- Подписаться на рассылку владельца страницы
- Заказать продукцию, предлагаемую владельцем страницы.
- Оплатить онлайн заказанную продукцию 


Задачи:

- Разработать, под ключ, сервис который обслуживает 3 категории пользователей:

- - "Дистрибьютер" - лицо занимающееся сетевым маркетингом
- - Потенциальные рекруты дистрибьютеров
- - Потенциальные покупатели продукции дистрибьютеров

- Реализовать прием платежей в автоматическом режиме (срок обслуживания аккаунта продлевается сразу после получения подтверждения оплаты от сервиса приема платежей
- Разработать дизайн в минималистичном дизайне и с упором на функциональность сервиса
- Составить инструкции по работе с админкой ЦМС
- Разместить сайт на хостинге, подключить систему приема платежей
